# 🚀 Pokémon Explorer

A modern and responsive React application that allows users to search and explore Pokémon data in real time using the PokéAPI.

The project focuses on building reusable React components, handling API integration efficiently, and creating a clean responsive user interface with modern frontend development practices.

---

## 🌐 Live Demo

👉 https://pokemonstate.netlify.app

---

## ✨ Features

* 🔍 Search Pokémon by name
* 📡 Real-time Pokémon API integration
* ⚡ Fast and responsive UI
* 🖼️ Pokémon image rendering
* 📊 Display stats, abilities, and types
* ⏳ Loading and error handling states
* ♻️ Reusable React components
* 📱 Mobile-friendly responsive design

---

## 📂 Installation & Setup

Clone the repository:

```bash
git clone https://github.com/ADEEB172/pokemon-explorer.git
```

Navigate to the project folder:

```bash
cd pokemon-explorer
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
